HARDWARE PROJECTS

Landmine Detection Robotic Vehicle

This project presents a landmine detection robotic vehicle equipped with GPS and GSM modules. The GPS ensures precise navigation across rough terrains, while the GSM enables remote control and real-time status updates. Using specialized sensors the vehicle scans and marks landmines accurately. This approach enhances operator safety and improves detection efficiency, offering a reliable solution to address the ongoing threat of landmines and promote global safety.

Image Processing Based Fire Detection

This project presents an image processing-based fire detection system using Arduino. The system utilizes a camera module and sensors to detect fire through visual cues and environmental parameters. The Arduino processes the captured images and sensor data to identify fire patterns with accuracy. Upon detection, it triggers an alert mechanism for timely response. This approach enhances early fire detection capabilities, ensuring faster emergency action and reducing potential damage.

SOFTWARE PROJECTS

Deep Neural Network for Fuzzy Automatic Melanoma Diagnosis

This project presents a deep neural network for fuzzy automatic melanoma diagnosis. It leverages advanced image processing and deep learning techniques to accurately detect melanoma, a severe form of skin cancer, from camera images. The fuzzy logic system refines diagnosis by handling uncertainty in skin lesion characteristics like benign and malignant. This approach enhances diagnostic accuracy and most importantly the patients can consolidate with doctors by filling a google form.
